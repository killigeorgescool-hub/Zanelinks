## 🎨 Personnalisation

### Changer les Couleurs
Dans `style.css`, modifiez les valeurs de couleur:
- Bleu principal: `#0066ff`, `#00d4ff`
- Fond: gradient `#0a0e27`, `#1a1f3a`

### Changer les Liens
Dans `index.html`, modifiez les URLs dans les balises `<a>` et ajoutez/supprimez des cartes selon vos besoins.

### Changer les Icônes
Les icônes viennent de Font Awesome. Consultez https://fontawesome.com pour plus d'options.

### Ajouter Plus de Liens
Dupliquez une section `.link-card` dans le HTML et mettez à jour:
- L'URL `href`
- L'icône avec la classe Font Awesome
- Le titre et la description

## 💡 Tips Supplémentaires

1. **Logo/Avatar**: Remplacez l'icône `<i class="fas fa-user-circle"></i>` par une image:
   ```html
   <img src="votre-photo.jpg" alt="Profile" class="profile-image">
   ```

2. **Domaine personnalisé**: Une fois déployé, vous pouvez acheter un domaine comme `zanekarter.link` et le connecter à votre site.

3. **Analytics**: Ajoutez Google Analytics pour tracker vos visiteurs.

4. **Email Signature**: Utilisez l'URL de ce site dans votre signature email.
